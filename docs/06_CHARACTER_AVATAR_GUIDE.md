# 06_CHARACTER_AVATAR_GUIDE — 조립형 캐릭터 아바타 가이드

Project ID: SIGNAL_PERSONAL
Project Name: Project Signal Personal
Status: active
Scope: 세로형 모바일 HTML/PWA 자동전투 개인 작업물

---

## 1. 캐릭터 표현 원칙

- 이미지 리소스 없이 HTML/CSS 조립형 아바타 우선
- 작은 모바일 화면에서 역할이 읽혀야 한다
- 직업은 색이 아니라 실루엣, 파츠, 자세로 구분한다
- 전열/후열 역할이 형태에서 읽혀야 한다
- 화려한 그래픽보다 전투 흐름과 직업 개성이 읽히는 것이 중요하다
- 캐릭터는 레고 블록처럼 파츠를 조립해 만든다

---

## 2. 아바타 파츠 개념

| 파츠 | 역할 |
|---|---|
| base | 몸통 |
| head | 머리 |
| weapon | 무기 |
| offhand | 방패 / 보조 장비 |
| aura | 오라 / 후광 |
| mark | 직업 상징 |
| stance | 전열 / 후열 자세 |

---

## 3. v0.1 기본 3직업 표현 방향

**전사 (warrior)**
- 단단한 전열 실루엣
- 검
- 안정적이고 넓은 자세

**사제 (priest)**
- 작은 지팡이
- 회복 빛 / 후광
- 부드러운 후열 인상

**궁수 (archer)**
- 활
- 후열 자세
- 날렵한 실루엣

---

## 4. 이후 기본 6직업 확장 방향

**수호자 (guardian)**
- 큰 방패
- 넓은 전열 실루엣
- 두껍고 묵직한 인상

**신관 (cleric)**
- 후광
- 축복 표식
- 안정적인 보조 인상

**교란꾼 (disruptor)**
- 날카로운 표식
- 흐트러뜨리는 인상
- 후열 / 측면 자세

---
