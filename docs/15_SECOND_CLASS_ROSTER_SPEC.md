## 0. 직업 총량 원칙 — 36종 MAX 완성형

Project Signal Personal의 직업 총량은 **36종 MAX**로 정의한다.

| 구간 | 티어 | 수 |
| -- | -- | -- |
| SR-01 ~ SR-06 | 기본 직업 | 6 |
| SR-07 ~ SR-21 | 1차 직업 | 15 |
| SR-22 ~ SR-36 | 2차 직업 | 15 |
| **합계** | | **36** |

- **36종은 Project Signal Personal의 직업 확장 상한선이다.**
- 36종을 넘는 신규 직업 추가는 원칙적으로 하지 않는다.
- 새 아이디어가 생기면 SR-01~SR-36 안에서 **흡수 / 교체 / 튜닝**한다.
- 이 원칙은 확장을 막기 위한 것이 아니라, **프로젝트가 완성 가능한 범위 안에서 마감**될 수 있도록 하는 울타리다.

## 1. 설계 목적 (2차 직업)

- **2차 직업은 "파티 운영 방식을 바꾸는 상위 직업"**이다. (강한 1차가 아님.)
- 1차 직업이 전투 **역할(role)**을 만드는 단계라면, 2차 직업은 파티의 **운영 축(axis)**을 확정하는 단계다.
- SR-22~24는 구현된 씨앗으로 유지, SR-25~30은 신규 6종, SR-31~36은 장기 잠금 씨앗.
- **아직 정식 해금은 하지 않는다.** 이 문서는 설계/세계선 정합 기록이며 코드/밸런스/해금을 확정하지 않는다.

## 2. 현재 코드 기준 정합성 (확정)

코드 기준 **기본 6**:
- 전사 / 수호자 / 궁수 / 사제 / 신관(cleric) / 교란꾼(trickster)

주의:
- **도적(rogue) / 마도(mage)는 기본 직업이 아니라 1차 직업이다.**
- 앞으로 문서와 설계는 이 **코드 기준**을 따른다. 슬롯 번호(SR) = 도감 번호 = 코드 기준 순서.

## 4. 신규 설계 대상 SR-25~SR-30 (초안)

각 항목은 공식 초안이며 수치/최종 로직은 미확정(후속 단계 결정).

### SR-25 검성 (key 후보: swordsaint)
- **조합:** 워든 + 전사 / **역할:** 전열 반격 / 처형 딜러 / **핵심 문법:** 결투 · 간파 · 반격 · 마무리
- 방향: 전열에 세우고 싶은 공격형 전열. 큰 공격을 받아낸 뒤 반격/강타. 저HP 적 처형 마무리.
  battle.js **counterStance orphan 로직**(데이터 미연결)을 반격 베이스로 재활용 후보 메모.
- 주의: 수호자/성황 같은 순수 방어가 아니라 "맞받아치는 검객".

### SR-26 구원자 (key 후보: redeemer)
- **조합:** 정화사 + 사제 / **역할:** 정화 / 위급 치유 / 소생 / **핵심 문법:** 정화 · 구원 · 생명선 · 전투당 1회 구제
- 방향: 정화사 상위형 + 소생/구제. 디버프 있으면 정화 우선, HP 매우 낮으면 위급 치유,
  전투당 1회 치명적 피해/기절 직전 아군을 HP 1로 버티게 하거나 즉시 소량 회복. HP 지속 + healReceivedBonus와 궁합.
- 주의: 단순 힐러 아님(보험형). **쉼터 완전회복/전투 후 HP 보정과 혼동 금지** → 전투 중 능력으로 제한.

### SR-27 천궁 (key 후보: skyarcher)
- **조합:** 추적자 + 궁수 / **역할:** 후열 표식 / 원거리 지휘 / 다중 저격 / **핵심 문법:** 표식 · 추적 · 하늘사격 · 우선순위 지정
- 방향: 추적자 표식 상위 확장 — 표식 대상 = 아군 공격 우선순위/추가 피해 지점. 저HP 적 저격 포함. 단일·다수 모두 사령탑.
- 주의: 파수궁 보복/후열 방어와 겹치지 않게. 용창 관통과 분리(천궁은 지정/표식/저격).

### SR-28 역병술사 (key 후보: plaguebringer)
- **조합:** 덫꾼 + 교란꾼 / **역할:** 독 / 디버프 확산 / 전장 약화 / **핵심 문법:** 독 · 확산 · 약화 · 감염
- 방향: 덫꾼 poison 칩+독 치명 상위 확장. 독 대상 공격 시 독/약화 전염. 소형 다수 강함, 보스엔 지속 약화(공/속 저하).
- 주의: 단순 독딜러 아님 — 전장을 무너뜨리는 컨트롤러.

### SR-29 무희 (key 후보: dancer)
- **조합:** 바드 + 신관 / **역할:** 파티 템포 / 축복 / 리듬 지휘 / **핵심 문법:** 춤 · 리듬 · 템포 · 축복
- 방향: 바드 리듬/템포 이중 구조 상위 확장 + 신관 축복(보호막/방어). 전열/후열·전황에 따라 다른 리듬.
- 주의: 바드처럼 완전 랜덤이면 안 됨(방향성). 이름은 부드러운 "무희".

### SR-30 결계장 (key 후보: wardkeeper)
- **조합:** 성벽 + 수호자 / **역할:** 지속 방어 / 보호막 / 진형 안정 / **핵심 문법:** 결계 · 보호막 · 결속 · 후열 보호
- 방향: 성황이 위기 1회를 막는 왕이라면 결계장은 지속 방벽. 전열 보호막/피해감소, 후열 피격 보호, 결속 강화, 경계도 높은 전투 안정.
- 주의: 성황의 "피해 1회 무효"와 겹치지 않게 — 지속 방어/보호막 중심.

### SR-25~30 요약 표
| 슬롯 | 이름 | 조합 | 역할 | 핵심 |
| -- | -- | -- | -- | -- |
| SR-25 | 검성 | 워든 + 전사 | 전열 반격 / 처형 딜러 | 결투, 간파, 반격, 마무리 |
| SR-26 | 구원자 | 정화사 + 사제 | 정화 / 위급 치유 / 소생 | 정화사 상위형 + 소생 |
| SR-27 | 천궁 | 추적자 + 궁수 | 표식 / 원거리 지휘 / 저격 | 표식, 추적, 하늘사격 |
| SR-28 | 역병술사 | 덫꾼 + 교란꾼 | 독 / 디버프 확산 | 독, 감염, 약화, 확산 |
| SR-29 | 무희 | 바드 + 신관 | 리듬 / 템포 / 축복 | 춤, 박자, 축복, 템포 조절 |

- SR-25~30은 아직 정식 해금하지 않는다. 우선 **데이터/도감/Dev 접근 씨앗**으로 확보, 전투 로직은 이후 2~3개 단위.
- 아바타는 루다 presentation-lab 쇼룸 후보 기준으로 나중에 이식.

## 5. SR-31~SR-36 장기 잠금 씨앗 (long-term locked seed)

> **중요:** SR-31~36은 "지금 만들 직업"이 아니라 **36종 MAX 체계를 닫기 위한 장기 잠금 씨앗**이다.
> - 지금 당장 구현하지 않는다(아바타/스킬/전투 로직/Dev 접근 모두 만들지 않는다).
> - 아래 레시피는 36종 MAX를 닫는 **1차 후보**이며, 실제 직업명/능력/해금 시점은 **테마 스테이지 설계 과정에서 조정**될 수 있다(절대 규칙 아님).
> - 스테이지/테마 설계 중 필요가 생길 때 **하나씩 깨우는** 방식으로 운용한다.

| 슬롯 | 재료 1 | 재료 2 | 상태 | 방향 메모 |
| -- | -- | -- | -- | -- |
| SR-31 | 파수궁 | 사제 | 장기 잠금 씨앗 | 후열 감시 / 보호 / 반응형 구원 |
| SR-32 | 성기사 | 신관 | 장기 잠금 씨앗 | 축복받은 기사 / 성전 / 전열 보호 |
| SR-33 | 금제 | 신관 | 장기 잠금 씨앗 | 금단 + 신성 / 봉인 / 위험한 축복 |
| SR-34 | 치유궁 | 전사 | 장기 잠금 씨앗 | 전장 의무 / 공격형 구조 / 공세 회복 |
| SR-35 | 수문장 | 사제 | 장기 잠금 씨앗 | 문 / 성역 / 방어 치유 / 진입 차단 |
| SR-36 | 성직자 | 전사 | 장기 잠금 씨앗 | 전투 성직자 / 기도와 무장의 결합 |

레시피 의도:
- 아직 2차 재료로 쓰이지 않은 1차 직업 6종(파수궁/성기사/금제/치유궁/수문장/성직자)을 장기 씨앗으로 **모두 보존**.
- 기본 직업 재료(사제·신관·전사 등) 사용도 전반적으로 균형 있게 유지.
- 단, 이 구성은 **절대 규칙이 아니라 장기 설계 후보**이며 테마 스테이지 필요에 따라 조정 가능.
