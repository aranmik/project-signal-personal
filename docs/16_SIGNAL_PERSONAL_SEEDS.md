## 0. Project Constitution

**핵심 문장**

> 세로 모바일 + 루트 선택 + 자동전투 + 합체 파티로 ‘이상한 던전의 판단 맛’을 만든다.

**해석**

- 플레이어는 직접 캐릭터를 격자에서 움직이지 않는다. 대신 **루트(갈림길)를 고르고**, 전투는 자동으로 흐른다.
- 재미의 중심은 손가락 조작이 아니라 **"이 길을 가도 되나?"라는 매 순간의 판단**이다.
- 합체 파티(직업 합성)·HP 계승·전리품 욕심이 그 판단에 무게를 싣는다.
- 세로 모바일 한 손 플레이에서 성립해야 한다.

**금지 오해**

- 그리드 이동 로그라이크의 복제가 **아니다**.
- 이상한 던전의 *외형*을 베끼는 것이 아니라, 그 **판단 감각을 모바일 세로 구조로 번역**하는 것이다.

### 프로젝트 헌법

- **카테고리:** Constitution
- **상태:** 기준선
- **핵심 문장:** 세로 모바일 + 루트 선택 + 자동전투 + 합체 파티로 ‘이상한 던전의 판단 맛’을 만든다.
- **플레이어 감정:** 내가 이동하지 않아도, 매 선택마다 "이걸 가도 되나?"를 느끼게 한다.
- **나중에 구현할 때 주의:** 이상한 던전의 외형이 아니라 판단 감각을 번역한다.
- **다음 액션:** 모든 본게임 티켓의 상위 기준으로 참조.

## 2. Combat / Status Seeds

전투 상태, 직업 정체성, 스킬 문법 씨앗.
(상태효과 시스템 자체를 다시 열 때 묶어서 검토하기 좋은 묶음입니다.)

### 천궁 — 대상표식 + 은신 + 추격

- **카테고리:** Combat / Job Identity
- **상태:** 씨앗
- **핵심 문장:** 천궁은 혼자 저격하는 직업이 아니라, 파티가 함께 찢을 대상을 지정하는 직업이다.
- **플레이어 감정:** "저놈을 모두가 같이 노린다."
- **나중에 구현할 때 주의:** 추적자의 조준과 천궁의 표식은 분리해야 한다(아래 "조준/표식 분리 문법" 참조).
- **다음 액션:** 상태효과 / 직업 스킬 확장 시 검토.

### 조준 / 표식 분리 문법

- **카테고리:** Combat / Status
- **상태:** 씨앗
- **핵심 문장:** 추적자 조준은 "내가 다음 턴 죽일 대상", 천궁 표식은 "파티가 함께 찢을 대상"이다.
- **플레이어 감정:** 같은 집중형 2턴 스킬이어도 역할 감각이 다르다.
- **나중에 구현할 때 주의:**
  - 둘 다 행위자의 1턴을 소모한다.
  - 은신 대상이 있으면 우선 사용할 수 있다.
  - 은신 대상이 조준 / 표식에 걸리면 은신 해제.
  - 추적자 조준은 다음 턴 저격 후 사라짐.
  - 천궁 표식은 다른 영웅도 1회 추가 피해를 얻고, 천궁이 직접 치면 즉시 사라짐.
- **다음 액션:** "은신 공용 상태효과"와 함께 설계.

### 은신 공용 상태효과

- **카테고리:** Combat / Status
- **상태:** 씨앗
- **핵심 문장:** 은신은 무적이 아니라 단일 타겟팅에서 빠지는 상태다.
- **플레이어 감정:** 사라졌지만, 완전히 안전하지는 않다.
- **나중에 구현할 때 주의:**
  - 아군 / 적 모두 사용 가능.
  - 단일 타겟 공격 대상에서 제외.
  - 광역 피해를 입으면 즉시 해제.
  - 조준 / 표식 당하면 즉시 해제.
  - 연출은 연기처럼 팡, 아바타 딤드 / 흐림.
  - 사용자 후보: 도적 · 천궁 · 추적자 · 암살 테마 몬스터 · 특정 정예 몬스터.
- **다음 액션:** 상태효과 시스템을 건드릴 때 후보로 재검토.

### One Loot Cause Map

- **카테고리:** Dev Observatory / Balance
- **상태:** Phase 3B 관측 결과

- **플레이어 감정:** 전리품을 들고 버티는 여정 자체가 위험해야 한다.
- **나중에 구현할 때 주의:** One Loot 확률을 완벽히 맞추는 것이 아니라 원인 지도를 확보하는 것이 목적이다.

### Deep Reward Pool / 전리품 후보 도감

- **카테고리:** Loot / Codex / Dev
- **상태:** 씨앗
- **핵심 문장:** 깊은 곳에서 나올 수 있는 전리품 후보와 보상 아이디어를 dev-accessible UI나 문서에서 볼 수 있게 한다.
