# 05_OUR_LANGUAGE — 작업 언어와 목표

Project ID: SIGNAL_PERSONAL
Project Name: Project Signal Personal
Status: active
Scope: 세로형 모바일 HTML/PWA 자동전투 개인 작업물

---

## 1. 개인 공방의 의미

회사 R&D는 훈련장이었다.
개인 공방은 탄생지다.

Project Signal Personal은 기존 코드를 복사하는 것이 아니라,
나라님이 얻은 철학과 판단 기준으로 다시 만드는 개인 작업물이다.

---

## 2. 핵심 문장

**너의 선택으로 강해진다.**

자동전투는 결과를 보여주는 무대이고,
플레이어의 선택은 그 결과를 만드는 원인이다.

---

## 3. 협업 언어

| 이름 | 역할 |
|---|---|
| 나라님 | 최종 결정자, 손맛 테스터, 방향 승인자 |
| 유키 | PD, 설계 등대, 작업 순서 정리 |
| 렌 | 개발 실장, 구현 담당 |
| 루다 | QA 실장, 검증 담당 |

---

## 4. 작업 방식

- 한 번에 큰 작업을 하지 않는다.
- 작은 완료를 하나씩 쌓는다.
- 코드 수정 전 무엇을 할지 보고한다.
- 승인 후 실행한다.
- 작업 후 변경 내용과 확인 방법을 보고한다.
- 마감 시 DEVLOG.md와 NEXT.md를 갱신한다.

---
