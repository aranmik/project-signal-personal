## 전달 규칙

### 말머리 형식

나라님 / 렌 / 유키 사이에 전달하는 모든 내용은 말머리로 발신자와 수신자를 명시한다.

```
[발신자 -> 수신자]
```

예시:
- `[나라 -> 렌]` — 나라님이 렌에게
- `[렌 -> 유키]` — 렌이 유키에게
- `[유키 -> 렌]` — 유키가 렌에게
- `[나라 -> 유키]` — 나라님이 유키에게

### 복붙 전달 형식

나라님이 렌 ↔ 유키 사이의 내용을 복붙으로 전달할 때,
전달 내용은 **하나의 코드 블록** 안에 담는다.

- 블록 밖에 부가 설명을 붙이지 않는다.
- 나라님이 그대로 복사해서 상대방에게 붙여넣을 수 있어야 한다.
- 렌이 유키에게 보내는 내용, 유키가 렌에게 보내는 내용 모두 동일하게 적용한다.
