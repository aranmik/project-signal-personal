## 2. Batch 01A — Watchbow / Purifier Official Alignment

**대상:** 파수궁 · 정화사

**목표**
- 공식 목표와 현재 구현 차이를 맞춘다.
- 신규 시스템을 크게 만들지 않는다.
- 전투 로직 변화는 최소화한다.
- 수치 밸런스보다 **문법 정합**이 우선이다.

### 파수궁 목표
현재:
- 후열 아군 피격 시 ATK ×1.0 반격 / 쿨 제한 없음

공식 목표:
- 후열 아군이 피해 입었을 때 발동
- 해당 아군을 공격한 공격자에게 즉시 반격
- 피해량은 파수궁 ATK의 **50%**
- 속도 게이지는 소모하지 않음
- 파수궁의 **일반 행동 1회 사이 최대 1번**
- 파수궁이 일반 행동하면 반격 가능 상태 **재충전**

검증:
- 후열 저격 몬스터 상대로 1회 반격 확인
- 같은 일반 행동 주기 안에서 반복 피격 시 추가 반격하지 않는지 확인
- 파수궁이 일반 행동 후 다시 반격 가능해지는지 확인
- 광역/다단히트에서 로그가 난잡하지 않은지 확인

### 정화사 목표
현재:
- 최저 다친/디버프 아군에게 정화+회복+보호막 동시 적용

공식 목표:
1. HP 30% 미만 아군이 있으면 치유 우선
2. 그 외 poison / atkDown / speedDown 등 디버프 또는 지속 피해 아군이 있으면 정화
3. 정화 성공 시 소량 보호막
4. 상태 이상이 없고 위급 대상도 없으면 기본 행동 또는 약한 회복 fallback 허용

검증:
- HP 30% 미만 아군이 있으면 정화보다 치유를 우선하는지
- poison 대상이 있고 위급 대상이 없으면 정화하는지
- 정화 성공 시 보호막이 들어가는지
- 상태 이상이 없는 전투에서 과도한 만능 힐러처럼 동작하지 않는지

**판정:** 01A는 다음 실제 개발 **1순위**.

## 3. Batch 01B — Cleric / Vanguard Identity Alignment

**대상:** 신관 · 선봉

**목표**
- 사제/신관 분리
- 신관 기반의 1차 직업 선봉 정체성 정렬
- 보호/축복/전열 방어 문법 강화

### 신관 목표
현재:
- 소량 회복 + guard / 사제와 heal grammar가 겹침

공식 목표:
- 기본 서포터
- 아군 2명에게 소량 보호막
- 또는 위협받는 아군에게 보호/축복 부여
- 회복 전문이 아니라 미래 피해를 막는 직업

검증:
- 사제와 체감이 분리되는지
- 신관이 힐러가 아니라 보호 서포터로 읽히는지
- 보호막이 전투 종료 후 클리어되는지
- HP 이월에서 사제와 선택 가치가 다르게 느껴지는지

### 선봉 목표
현재:
- 전열 AoE + 미량 회복

공식 목표:
- 전열 아군 방어 증가
- 전열 적 전체 낮은 피해
- 회복보다 진군/방어/전열 압박으로 정리

검증:
- 전열 아군이 더 버티는지
- 적 전열을 압박하는 체감이 있는지
- 후열 보호와는 구분되는지
- 사제/성직자/정화사와 역할이 겹치지 않는지

**판정:** 01A 완료 후 진행.

## 4. Batch 01C — Trapper / Bard Personality Alignment

**대상:** 덫꾼 · 바드

**목표**
- 체감 변화가 큰 직업 개성 조정
- poison / crit / random table 문법을 실제 전투에 적용
- 로그/FX 읽힘을 고려해야 하므로 01A/01B 이후 진행

### 덫꾼 목표
현재:
- 중독 poison 2턴

공식 목표:
- 임의의 적 2명 중독
- 중독 대상 공격 시 치명 보정 또는 확정 치명 후보
- 단순 DoT가 아니라 독 표식/치명판 서포터

검증:
- 다수전에서 중독 2명 부여 확인
- 중독 대상 공격 시 치명 연계가 읽히는지
- 단일 보스전에서 과도하게 강하지 않은지
- 정화/독 저항/장갑 테마와 연결 가능한지

**판정:** poison과 crit 연동이 필요하므로 01C로 보류.

### 바드 목표
현재:
- rhythmTempo 고정 패턴 / 아군 공↑ + 치명 예약 · 적 속↓ + 드레인

공식 목표:
- 완전 랜덤 음유시인
- 아군/적 선택 랜덤
- 대상 랜덤
- 효과 랜덤
- 아군이면 공격력 증가 또는 치명 확률 증가
- 적이면 속도 감소 또는 행동 게이지 감소

검증:
- 랜덤이지만 "무슨 일이 일어났는지" 읽히는지
- 짧은 전투에서 너무 무가치하지 않은지
- 긴 전투에서 너무 강하지 않은지
- 보스전에서 무작위 효과가 지나치게 흔들지 않는지

**판정:** 로그/FX 읽힘이 중요하므로 01C로 보류.

## 5. 개발 순서 결론

1. **Batch 01A** — 파수궁 · 정화사
2. **Batch 01B** — 신관 · 선봉
3. **Batch 01C** — 덫꾼 · 바드

**이유**
- 01A는 공식 목표와 현재 구현 차이를 맞추는 **저위험 정렬**.
- 01B는 기본 직업/1차 직업 **정체성 조정**.
- 01C는 체감 변화가 크고 poison/crit/random table 등 **추가 검증이 필요한 조정**.
